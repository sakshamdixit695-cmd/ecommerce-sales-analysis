# ecommerce-sales-analysis
SQL + Python analysis of e-commerce sales and customer behavior
## Overview
This project analyzes transactional e-commerce data using **PostgreSQL (SQL)** and **Python** to understand revenue patterns, customer behavior, and purchasing trends.

## Tools & Technologies
- PostgreSQL (SQL, CTEs, window functions)
- Python (pandas, matplotlib, seaborn)
- Google Colab

## Project Workflow
1. Validated and analyzed raw transactional data using SQL
2. Performed customer segmentation and revenue analysis
3. Conducted time-based and demographic analysis
4. Created visualizations and derived business insights

## Key Insights
- Revenue is evenly distributed across product categories and countries
- Customers aged 46–60 are the highest revenue contributors
- Monthly sales show stable trends with seasonal peaks
- Payment methods show balanced adoption with no single dependency

## Files
- `sql/ecommerce_analysis.sql` → SQL queries
- `notebooks/Ecommerce_SQL_Python_Analysis.ipynb` → Python analysis & visuals
